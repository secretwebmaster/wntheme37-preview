## 安裝指南
- 將模板文件上傳到 MacCMS V10 的 `template` 目錄下
- 確認模板目錄名稱爲 wntheme37
- 登錄後台，進入 **系統 -> 網站參數配置 -> 網站模板**
- 選擇 wntheme37
- 模板目錄輸入 html
- 保存後訪問1次域名後，模板會自動安裝
- 首次打開頁面顯示亂碼是正常，刷新頁面查看效果
